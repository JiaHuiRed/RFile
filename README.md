# RFile

**macOS 风格 Windows 文件管理器**

`V0.2.0` · `Windows 10/11` · `Python 3.10+`

</div>

---

RFile（Red File Manager）是一款以 macOS Finder 为设计参考的 Windows 文件管理器，纯 Python 编写，提供毛玻璃效果、多标签页、空格快速预览等 Windows 资源管理器所没有的体验。

## 特性

| | 功能 | 说明 |
|---|---|---|
| 🪟 | **毛玻璃效果** | 标题栏与侧边栏实时截图模糊，随壁纸变化 |
| 🗂 | **多标签页** | 同时浏览多个目录，独立历史记录 |
| 🔍 | **Quick Look** | 空格键预览图片/文本/代码/文件夹，无需打开 |
| 🍞 | **面包屑路径栏** | 可点击跳转的路径段，双击切换为手动输入 |
| 🎨 | **四套主题** | 深蓝 / 护眼绿 / 暖黄 / 日间，一键切换 |
| 🚦 | **macOS 交通灯** | 红关闭 / 黄托盘 / 绿最大化（再次点击还原） |
| 📁 | **完整文件操作** | 复制、剪切、粘贴、移动、重命名、删除、新建文件夹 |
| 🔎 | **文件属性** | 查看大小、创建时间、修改时间，文件夹递归计算总大小 |
| 📂 | **打开方式** | 调用 Windows 原生"打开方式"对话框 |
| 📌 | **窗口置顶** | 切换始终显示在最前 |
| 👻 | **隐藏文件** | 右键菜单开关显示 `.` 开头的隐藏文件 |
| 🖱 | **无边框缩放** | 拖拽窗口边缘/四角自由调整大小 |
| ⭐ | **常用文件夹** | 自动统计访问频次，侧边栏展示最常用的 3 个文件夹 |

## 快速开始

**直接运行（推荐）**

下载 `dist/RFile.exe`，双击即用，无需安装 Python。

**从源码运行**

```bash
pip install Pillow
python RFile.py
```

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `空格` | Quick Look 预览（再按关闭） |
| `Enter` | 打开文件/进入文件夹 |
| `BackSpace` | 返回上级 |
| `F5` | 刷新 |
| `F2` | 重命名 |
| `Delete` | 删除（有确认弹窗） |
| `Ctrl+T` | 新建标签页 |
| `Ctrl+W` | 关闭当前标签 |
| `Ctrl+Tab` | 切换到下一标签 |
| `Ctrl+Shift+Tab` | 切换到上一标签 |
| `Ctrl+Shift+N` | 新建文件夹 |
| `Ctrl+C / X / V` | 复制 / 剪切 / 粘贴 |
| `Esc` | 关闭预览窗口 / 退出 |
